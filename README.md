## What's Inside

This repository contains cleaned, structured, and continuously expanded notes covering:

- **LLM Fundamentals** — How language models work, their strengths and limitations
- **AI Agent Patterns** — ReAct, tool use, memory, planning, and multi-agent architectures
- **Model Selection** — Comparing models by size, capability, cost, and deployment constraints
- **Prompt Engineering** — Zero-shot, few-shot, chain-of-thought, and advanced prompting techniques
- **Architecture Impact** — How integrating LLMs changes system design, scalability, and reliability
