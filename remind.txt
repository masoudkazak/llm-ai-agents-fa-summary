    - LangGraph {
        لانگ‌گراف (LangGraph) یک چارچوب (Framework) برای ساخت ایجنت‌های هوش مصنوعی مبتنی بر مدل‌های زبانی بزرگ است که رفتار ایجنت را به‌صورت یک گراف از حالت‌ها و جریان‌ها مدل‌سازی می‌کند. در LangGraph، هر نود نمایانگر یک مرحله یا عمل (مثل فراخوانی LLM، استفاده از ابزار، یا تصمیم‌گیری) و edges مسیر انتقال بین این مراحل هستند. این رویکرد امکان ساخت ایجنت‌های چندمرحله‌ای، پایدار (Stateful)، قابل کنترل و قابل تکرار را فراهم می‌کند.
    }
    
    - `state` is like a container that stores and passes data between different parts of our workflow. Each node receives and returns a state object.

        -> llm = ChatOpenAI(model="gpt-4o-mini", temperature=0, streaming=True)
        -> class AgentState(TypedDict):

        -> tavily_search_tool = TavilySearchResults(max_results=3)

        -> def make_call_model_with_tools(tools: list):
        ->     def call_model_with_tools(state: AgentState):
        ->         messages = state["messages"]
        ->         model_with_tools = llm.bind_tools(tools)
        ->         response = model_with_tools.invoke(messages)
        ->         return {"message": [response]}
        ->     return call_model_with_tools
        -> 
        -> def should_continue(state: AgentState) -> Literal["action", "__end__"]:
        ->     last_message = state["messages"][-1]
        ->     if isinstance(last_message, AIMessage) and hasattr(last_message, "tool_calls") and last_message.tool_calls:
        ->         return "action"
        ->     else:
        ->         return END
        -> 
        -> def build_graph_one_tool(tools_list):
        ->     tool_node = ToolNode(tools_list)
        ->     call_node_fn = make_call_model_with_tools(tools_list)
        ->     
        ->     graph_one_tool = StateGraph(AgentState)
        ->     graph_one_tool.add_node("agent", call_node_fn)
        ->     graph_one_tool.add_node("action", tool_node)
        ->     
        ->     graph_one_tool.set_entry_point("agent")

        ->     graph_one_tool.add_conditional_edges(
        ->         "agent",
        ->         should_continue,
        ->         {"action": "action", END:END},
        ->     )
        ->     graph_one_tool.add_edge("action", "agent")
        ->     app = graph_one_tool.compile()
        ->     return app

        -> def app_call(app, messages):
        ->     initial_state = {"messages": [HumanMessage(content=messages)]}
        ->     final_state = app.invoke(initial_state)
        ->     return final_state["messages"][-1].content, final_state
        -> 
        -> messages = "Lorem ipsum..."
        -> app = build_graph_one_tool([tavily_search_tool])
